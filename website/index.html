Anchor

Classroom focus and device management, made by teachers and free for everyone. A teacher starts a focus session for a class; for its duration each student's device softly enforces a teacher-defined allowlist, and off-task activity surfaces live on the teacher's dashboard.

For teachers Source code

How it works

01

Start a session

A teacher opens Anchor and starts a focus session for the class, choosing an allowlist of permitted apps and sites.

02

Devices focus

For the session's duration, each student's device softly enforces the allowlist — no installs, no lockdown when class ends.

03

See off-task activity

Off-list activity surfaces live on the teacher's dashboard so the teacher can nudge a student back on track.

Free & open source

Made by teachers, free for everyone

Anchor is GPL-3.0 licensed and self-hostable. Each school runs its own backend; Plink Labs operates none and receives no student data.
